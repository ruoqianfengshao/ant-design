# 排列

行内组合组件，为同类型子组件提供简单的排列方式。

## Api

| 参数 | 说明   |  类型    | 默认值     |
| ---- | ---------- | -------- |
style | 自定义样式 | `CSSProperties` | {}
space | 组件两两之间间隔 | `string` | 16px
itemStyle | 组件最右边的右边距 | 0
align | 排列方式 | 'horizontal', 'vertical' | 'horizontal'
prefixCls | 组件前缀 | `string` | 'antd'
children | 子组件 | React.ReactNodeArray | []
