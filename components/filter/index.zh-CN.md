多组筛选条件集合

## 何时使用

针对

## API

| 参数 | 说明   |  类型    | 默认值     |
| ---- | ---------- | -------- |
fields | 筛选表单项集合 | `FieldItemProps[]` | []
firstLineLimit | 第一行显示表单项个数 | `number` | 3
customCount | 自定义保存筛选条件的个数 | 1
prefixCls | 组件前缀 | `string` | 'antd'
